# MirrorMan Roadmap

This document outlines the planned features and improvements for future versions of MirrorMan.

---

## v0.3 - UI & Usability Improvements

**Planned: Q2 2026**

- [ ] Add system tray icon for background operation
- [ ] Implement mirror status indicator in sidebar
- [ ] Add keyboard shortcuts for common actions
- [ ] Improve mirror list virtualization for large datasets
- [ ] Add dark/light theme toggle in settings

---

## v0.4 - Mirror Management

**Planned: Q3 2026**

- [ ] Support custom mirror URLs (not just Arch API)
- [ ] Add mirror backup and restore functionality
- [ ] Implement scheduled automatic mirror testing
- [ ] Add mirror health monitoring with alerts
- [ ] Export/import mirror configurations

---

## v0.5 - Repository Features

**Planned: Q4 2026**

- [ ] Add repository priority ordering
- [ ] Implement repository group management
- [ ] Add custom repository support (user-defined repos)
- [ ] Repository sync progress in notification area
- [ ] Detect and warn about conflicting packages across repos

---

## v0.6 - Advanced Features

**Planned: Q1 2027**

- [ ] Mirror failover automatic switching
- [ ] Bandwidth usage tracking per mirror
- [ ] Historical mirror performance graphs
- [ ] Multi-distribution support (Manjaro, EndeavourOS)
- [ ] Plugin system for extensions

---

## v1.0 - Stable Release

**Planned: Q2 2027**

- [ ] Complete translation to Persian (fa)
- [ ] Complete translation to other languages
- [ ] Flatpak support
- [ ] Comprehensive test suite
- [ ] Release on Parch Linux official repositories
- [ ] Full documentation and user guide

---
